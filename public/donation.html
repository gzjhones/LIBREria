La LIBREria es una proyecto de Código abierto desarrollado por la comunidad. Tiene como objetivo permitir a la población paceña el poder descargar libros o subir su propio material, siempre y cuando este se encuentre bajo una licencia abierta.
Ya que el proyecto no tiene un financiador pues permanece a la comunidad de Código abierto, le solicitamos que pueda hacer una donación al proyecto, para que este pueda expanderse, ser autosostenible y llegar a otros lugares del país y el mundo. Contactarse a:
